# Client carousel images

Save 12 client logos or collaboration images here:

```
client-01.png
client-02.png
...
client-12.png
```

Paths are referenced as `assets/clients/client-01.png` etc. in `js/preferences.js` and can be edited live in Tweakpane under **Client items**.

Recommended: square or 16:10 PNG/WebP with transparent background for logos.
